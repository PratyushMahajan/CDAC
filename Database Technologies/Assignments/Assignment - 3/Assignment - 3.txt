Overview of SQL.

1. Does ANSI recognize the data type DATE?  
   - Yes, ANSI SQL (American National Standards Institute) recognizes the DATE data type. It is used to store date values, usually in the format YYYY-MM-DD, across different SQL databases.

2. Which subdivision of SQL is used to insert values in tables?  
   - DML (Data Manipulation Language)  
   - DML is the part of SQL used to modify data in tables. The `INSERT` command, which adds new rows to a table, is one of the main DML operations.
